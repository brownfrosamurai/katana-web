# Overview

This portfolio site showcases my work and writing. Built with Next.js, Tailwind CSS, and Velite for content—MDX powers both the blog and project pages.

![Portfolio homepage](https://images.unsplash.com/photo-1467232004584-a241de8bcf5d?w=800&q=80)

## Approach

Dark-first design with a focus on typography and readability. Velite processes MDX content at build time for fast static pages. The structure supports easy addition of new projects and blog posts.

## Features

- MDX-powered blog and projects
- Dark-first design
- Responsive layout
