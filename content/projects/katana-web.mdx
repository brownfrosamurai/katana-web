# Overview

A personal portfolio that showcases my work and writing. Built with Next.js, TypeScript, and Tailwind CSS. Features MDX-powered blog and project pages, Velite for content management, and Framer Motion for animations.

![Portfolio homepage](https://images.unsplash.com/photo-1467232004584-a241de8bcf5d?w=800&q=80)

## Approach

Velite processes MDX content at build time for fast static pages. Add blog posts to content/blog/ and projects to content/projects/, then run velite to regenerate. Uses next-themes for dark mode and Framer Motion for smooth transitions.

## Features

- MDX-powered blog and projects
- Dark mode with next-themes
- Responsive layout
