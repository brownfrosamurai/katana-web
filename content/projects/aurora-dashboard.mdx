# Overview

A real-time analytics dashboard built for product teams. Features interactive data visualization, custom report builders, and team collaboration.

![Aurora Dashboard](https://images.unsplash.com/photo-1551288049-bebda4e38f71?w=800&q=80)

## Approach

Designed for product and growth teams who need to monitor metrics in real time. Custom report builders let users define their own views. Data exports to PDF and CSV for sharing.

## Features

- Live data streaming
- Custom metric definitions
- Export to PDF and CSV
