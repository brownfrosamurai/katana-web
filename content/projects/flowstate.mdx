# Overview

A design system and component library built for rapid prototyping and consistent product experiences. Includes accessible components, theme customization, and dark mode support.

![Flowstate component library](https://images.unsplash.com/photo-1561070791-2526d30994b5?w=800&q=80)

## Approach

Started with an audit of existing UI patterns, then codified them into reusable React components. Storybook serves as both documentation and a design review environment. The Figma kit keeps designers and developers aligned.

## Features

- Accessible components
- Theme customization
- Dark mode support
