# Overview

A fast, modern web framework for building scalable applications with focus on performance and developer experience. TypeScript-first, edge and serverless ready.

![Nitro framework](https://images.unsplash.com/photo-1498050108023-c5249f4df085?w=800&q=80)

## Approach

Optimized for edge and serverless deployments with minimal cold start times. The CLI provides scaffolding, dev server, and production builds. Full TypeScript support throughout.

## Features

- Lightning-fast cold starts
- Edge and serverless ready
- TypeScript first
