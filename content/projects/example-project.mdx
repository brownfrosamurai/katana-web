# Overview

This is a social media platform built for entrepreneurs to connect, share ideas, and collaborate on ventures. The project combined thoughtful UX design with a modern tech stack to deliver a seamless experience across web and mobile.

![Bizz Buzz mobile app on device](https://images.unsplash.com/photo-1512941937669-90a1b58e69e3?w=800&q=80)

## Approach

We started with user research and competitive analysis to understand the market. Wireframes and prototypes were iterated in Figma before moving to development. The final product leverages Next.js for performance and Tailwind CSS for a consistent design system.

![Desktop and mobile views](https://images.unsplash.com/photo-1496181133206-80ce9b88a853?w=800&q=80)

## Outcome

The platform launched successfully with strong engagement in the first quarter. Key features include real-time messaging, project collaboration tools, and an algorithmic feed tailored to entrepreneurial content.
