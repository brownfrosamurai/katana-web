# Overview

A companion app for fitness enthusiasts to track workouts, join community challenges, and visualize progress over time. The app combines activity logging with social features to keep users motivated.

![Pulse Mobile on device](https://images.unsplash.com/photo-1571019614242-c5c5dee9f50b?w=800&q=80)

## Approach

We prioritized intuitive activity logging and a clean, motivating interface. Social challenges and leaderboards encourage consistent engagement. The backend handles real-time sync and workout analytics.

## Features

- Activity logging
- Social challenges
- Progress insights
