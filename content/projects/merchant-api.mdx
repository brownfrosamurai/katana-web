# Overview

RESTful payment processing API designed for e-commerce platforms. Includes webhook delivery, idempotency keys, and a full sandbox environment for integration testing.

![API architecture](https://images.unsplash.com/photo-1558494949-ef010cbdcc31?w=800&q=80)

## Approach

Built with reliability in mind—idempotency keys prevent duplicate charges, and webhook retry logic ensures delivery. The sandbox environment lets developers test integrations without real transactions.

## Features

- Card and alternative payments
- Webhook retry logic
- Developer sandbox
